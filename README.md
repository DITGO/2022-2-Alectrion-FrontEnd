## Sobre

Alectrion é um sistema de controle e gerenciamento de inventário e ordens de serviços de TI.

Aplicação disponível em: [link da aplicação](https://alectrion-2022.herokuapp.com/)

## Requisitos

- node (versão 16.10.0 ou superiores)
- yarn (versão 1.22.18)

## Instalação

1. Clone o projeto 
> git clone https://github.com/fga-eps-mds/2022-1-Alectrion-FrontEnd
2. Entre na pasta do projeto
> cd 2022-1-Alectrion-FrontEnd
5. instale as depencencias
> yarn
6. Crie um arquivo .env da mesma forma do arquivo .env.example 
7. execute o projeto
> yarn dev
8. inicie a aplicação na porta 3000
